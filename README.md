# 📚 Flashcards de Inglés

Aplicación web para practicar inglés mediante distintos modos de aprendizaje:

Características

* Traducción de palabras inglés → español.
* Preguntas de opción múltiple.
* Listado completo de vocabulario.
* Sistema de vidas en el modo opción múltiple.
* Reinicio de partida al perder todas las vidas.
* Diseño adaptable para dispositivos móviles.
* Compatible con GitHub Pages.

Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* JSON

Estructura del proyecto

```text
Flashcards/
│
├── index.html
├── translate.html
├── multiple_choice.html
├── vocabulario.html
│
├── data/
│   ├── vocabulario.json
│   └── frases.json
│
├── js/
│   ├── translate.js
│   ├── multiple_choice.js
│   └── vocabulario.js
│
└── static/
    └── style.css
```

Uso

1. Abrir el sitio web.
2. Elegir un modo de estudio.
3. Practicar vocabulario y traducciones.
4. Mejorar el aprendizaje mediante la repetición.
